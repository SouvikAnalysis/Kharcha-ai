# Purpose

This document defines every analytics event generated by Kharcha.

Analytics helps us:

- Understand user behavior
- Improve user experience
- Measure feature adoption
- Evaluate AI effectiveness
- Increase user retention
- Make product decisions using data

Analytics should never collect sensitive financial information unnecessarily.

---

# Analytics Principles

Every tracked event must answer at least one question.

If an event cannot answer a business or product question,

it should not be tracked.

---

# Event Naming Convention

```

```
