# Objective

This document defines all the functional requirements of Kharcha.

The goal is to clearly identify:

- MVP Features
- Future Features
- Premium Features
- Admin Features
- Out of Scope Features

Every feature must follow Kharcha's Golden Rule:

> "Does this help users understand or control their money more easily?"

If the answer is NO, the feature should not be included in the MVP.

## Expense Management

Priority: High

Users should be able to:

- Add Expense

- Delete Expense
- Search Expense
- Filter Expenses
- Add Notes
- Select Category

Expense entry should take less than **5 seconds**.

# Out Of Scope (Version 1)

The following features will NOT be developed during the MVP.

- Bank Integration
- UPI Auto Sync
- Credit Card Analysis
- Investment Tracking
- Loan Management
- Tax Calculator
- Insurance Management

These may be considered in future releases.

---

# Success Criteria

The MVP will be considered successful if users can:

- Record an expense within 5 seconds.
- Understand where their money goes.
- Stay within budget.
- Receive meaningful reminders.
- Build a daily expense tracking habit.

---

# Golden Rule

Every feature must answer:

"Does this help users understand or control their money more easily?"

If NO,

The feature will not be added.
